hola

¡ Hola !

Este "Lab" enseña la sintasis basica de funciones en Ruby.

Abra un terminal en este directorio

cd 00_hola

Este directorio es el punto de entrada del ejercicio. Contiene un fichero "spec" y deberas añadir un fichero Ruby " standard " y conseguir que pase los " specs " previstos en el ejercicio.

Ejecute los test

rake

Vea como falla

Podra ver un error. ¡ No se asuste ! Trate de leer y entener lo que la computadora quiere decirle. En algun sitio de la primera linea deberia decir algo como esto

no such file to load -- test-first-teaching/hola/hola (LoadError)

Significa que esta buscando un fichero llamado hola.rb y no puede encontrarlo.

Cree el fichero: hola.rb

Abra hola.rb en un editor de texto. Grabelo. Ejecute el test de nuevo.

rake

Vea como falla

Ahora, deberia ver un error como este:

the hola function
  says hola (FAILED - 1)

Failures:

  1) the hola function says hola
     Failure/Error: hola.should == "hola!"
     NameError:
       undefined local variable or method `hola' for #<RSpec::Core::ExampleGroup::Nested_1:0x000001009b8808>
     # ./hola/hola_spec.rb:5:in `block (2 levels) in <top (required)>'

Cree la funcion "hola"

Arregle este error abriendo hola.rb y creando una funcion vacia:

def hola
end

Grábelo. Ejecute el test de nuevo.

Mire como falla

Ahora, deberia ver un error como este:

the hola function
  says hola (FAILED - 1)

Failures:

  1) the hola function says hola
     Failure/Error: hola().should == "hola!"
       expected: "hola!"
            got: nil (using ==)
     # ./hola/hola_spec.rb:5:in `block (2 levels) in <top (required)>'

Esto significa que aunque ha encontrado el fichero, y tambien ha encontado la funcion, pero ¡ esta funcion no devuelve nada ! ("nil" es la forma que tiene Ruby de decir "no hay nada".)

Haga que devuelva algo

Dentro de la funcion "hola", ponga una sola linea que contenga un cadena de texto que no sea "¡Hola!". (Aqui estamos simulando que comete un error pequeño, para que pueda ver lo que parece el mensaje de error.)

def hola
  "¡guau!"
end

Grabelo. Ejecute el test otra vez.

Mire como falla

Ahora debería poder ver un error como este:

1) the hola function says Hola
   Failure/Error: hola().should == "¡ Hola !"
     expected: "¡ Hola !"
          got: "¡ guau !" (using ==)
   # ./hola/hola_spec.rb:5:in `block (2 levels) in <top (required)>'

Corrijalo cambiando "¡guau!" por "¡Hola!". Grabelo. Ejecute el test de nuevo.

¡ Mire como pasa bien ahora !

the hola function
  says hola

the saludar function
  says hola to someone (FAILED - 1)

En orden a que la siguiente funcion para el test correctamente, deberia ser capaz de aceptar un argumento.

def saludar(alguien)
  "Hola, #{alguien}!"
end

Tests

hola_spec.rb 
require "hola"

describe "la funcion hola" do
  it "dice hola" do
    hola.should == "¡ Hola !"
  end
end

describe "la funcion saludar" do
  it "dice hola a alguien" do
    saludar("Alicia").should == "! Hola, Alicia !"
  end

  it "dice hola a alguien mas" do
    saludar("Roberto").should == "¡ Hola, Roberto !"
  end
end
